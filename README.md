# Research Paper Intelligence Platform

An AI-powered system for intelligent research paper analysis and discovery, built with RAG (Retrieval-Augmented Generation) and multi-agent architecture.

## Tech Stack
- **LLM**: Groq (Llama 3.3 70B) / Google Gemini — Free APIs
- **Embeddings**: Sentence Transformers / HuggingFace
- **Vector DB**: ChromaDB
- **Orchestration**: LangGraph (Multi-Agent)
- **Backend**: FastAPI
- **Frontend**: Streamlit
- **Fine-tuning**: LoRA via HuggingFace PEFT

## Setup
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
cp .env.example .env  # Add your API keys
```
